## paths (and extras)
* autorun       -> ~/.config/autorun
* awesome       -> ~/.config/awesome
* nitrogen      -> ~/.config/nitrogen
* rofi          -> ~/.config/rofi
* dwm           -> /usr/local/bin
* urxvt         -> ~/.Xresources
* misc theme    -> /usr/share/themes
* scripts       -> /usr/local/bin
